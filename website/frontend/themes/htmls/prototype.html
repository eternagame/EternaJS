Eterna Script

What is Eterna Script?

Eterna Script is an extensive JavaScript library that provides functionality for RNA folding. It provides the user with flexible tools to design their own RNA folding algorithms by giving them pre-defined functions that serve as building blocks for the actual algorithm.
One of the several good things about Eterna Script is that it can be used by both Eterna developers, to write scripts to improve the Eterna experience itself, and Eterna players, to design RNA Folding algorithms to solve puzzles. This flexibility is at the heart of Eterna Script, so as to allow it to augment the process of translating an idea into code and not prove to be a hindrance in the developmental process.
Eterna Script also serves as a core tool for players interested in participating in RNA Algorithm Wars, by providing them with a platform to design their RNA Folding algorithms on.

Why Eterna Script?

When Eterna Developers set out to identify a comprehensive RNA Folding library, that was not simply functionally sound, but was also easy to use, they pulled a blank. There are several RNA Folding libraries available but where some lack in functionality, others miss out on intuitiveness and ease of use. To fill this void and in the process provide Eterna players and algorithm designers with a comprehensive library that is intuitive to use, Eterna developers built the Eterna Script library on top of one of the most programming popular languages in the world, JavaScript.
What this entails is that the Eterna Script user is not simply limited to the Eterna Script library functionality but can also choose to develop in pure JavaScript, according to his/her needs. Eterna Script is designed with the aim of becoming a useful tool to design RNA Folding algorithms that can be used to tackle Eterna puzzles.

What is the RNA-AW?

The RNA Algorithm Wars (RNA-AW) is a great opportunity for Eterna players to compete with each other and collectively promote a healthy competitive environment for fully understanding the concepts that go into RNA folding. Since RNA-AW focuses on designing the algorithms that govern RNA folding, it encourages participants to work out algorithmic techniques to solve RNA folding puzzles and discourages result oriented techniques tailored to specific puzzles, like brute forcing and hit-and-trial, to a large extent.
The Eterna RNA-AW is powered by Eterna's homebrew language Eterna Script, the most comprehensive and advanced RNA folding library, written in Javascript, designed especially for Eterna. It gives the user enough flexibility to realize his ideas but at the same time, also takes care that the user does not feel lost or intimidated.

How will I be ranked?

Eterna RNA-AW rankings are based on player ratings. The higher a player's rating, the higher his rank.

Overview of the Eterna RNA-AW Rating System

The ranking system of Eterna RNA-AW, although slightly complicated, is designed for accuracy and efficiency. Submitted algorithms and puzzles are rated relatively on their success rate against each other. The base rating (or average rating) for an algorithm and a puzzle when it starts off is 1200. This rating is improved everytime the algorithm solves a puzzle (and improves a lot if it solves a puzzle of a higher rating than its own!) and drops if it fails to solve a puzzle (and drops a lot if it fails to solve a lower rated puzzle). Puzzle ratings are also updated in a similar fashion.
A general rule of thumb is that a difference of more than 200 points in rating corresponds to the higher rated algorithm or puzzle winning against the lower rated one with an estimated success percentage of 75%.
